#### Step 2-2: 교육 중심 덮어쓰기 개발 지시
```
서대리님에게 전달할 교육 최적화 지시 템플릿:

=== V2.0 교육 미션 ===
📋 교육 목표: [구체적 학습 목표]
🎯 미션: 기존 V1.0 파일을 개선된 V2.0으로 업그레이드

📁 대상 파일: Z:\html\[정확한 파일명] (기존 V1.0 덮어쓰기)
🔄 교육 방식: 분석 → 개선 → 덮어쓰기 → 즉시 확인

📚 학습 단계:
1. 분석: 현재 Z:\html\[파일명] 구조 파악
2. 참고: backupv1.0\[파일명]과 비교 분석
3. 개선: 동일 기능을 더 깔끔한 코드로 작성
4. 교체: 기존 파일을 개선된 V2.0으로 덮어쓰기

📝 개선 요구사항:
- 100% 동일한 기능 유지 (절대 조건)
- 코드 품질 향상 (가독성, 유지보수성)
- 현대적 웹 기술 적용 (HTML5, CSS3, ES6+)
- 성능 최적화 적용

🎨 디자인 가이드:
- 기존 V1.0과 동일한 디자인 유지
- backupv1.0/css/ 스타일 참고

📱 반응형 요구사항:
- 기존과 동일한 반응형 동작 보장

⏰ 예상 완료 시간: [시간]
✅ 완료 보고: "[파일명] V2.0 업그레이드 완료, 덮어쓰기 저장됨" 

🎓 교육 포인트: 
- 실제 운영 사이트 직접 개선하는 실무 경험
- Before & After 즉시 비교 가능
- 온라인에서 즉시 결과 확인 가능
- 실수해도 backupv1.0에서 언제든 복구 가능

🚀 레벨업 시스템:
현재 레벨: [레벨] → 이번 미션 완료시 다음 레벨 도전!
